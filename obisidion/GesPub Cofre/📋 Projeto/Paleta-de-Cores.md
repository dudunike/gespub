# Paleta de Cores

## Resumo
Este documento especifica a identidade cromática e o sistema de cores do GESPUB.AI, definindo os tokens e as regras de aplicação em toda a interface do usuário.

## Conteúdo
As cores do GESPUB.AI foram projetadas para criar uma estética moderna, limpa e profissional, utilizando tons de roxo (brand) combinados com superfícies neutras de alta legibilidade.

### Cores Principais

#### 1. Marca (Brand) - Roxo
*   **Brand 500 (Primária):** `#7C3AED` - Usada para botões principais, links ativos, destaques visuais e foco.
*   **Brand 700:** `#5B21B6` - Usada para estados hover em botões primários.
*   **Brand 100:** `#DDD6FE` - Usada para bordas suaves e seleções.
*   **Brand 50:** `#EDE9FE` - Fundo de itens ativos ou destacados.

#### 2. Superfícies (Surface)
*   **Fundo da Aplicação (bg):** `#F4F4F6` - Fundo neutro de baixa saturação para reduzir o cansaço visual.
*   **Card / Painéis:** `#FFFFFF` - Branco puro para destacar os blocos de conteúdo e tabelas.

#### 3. Textos (Typography)
*   **Texto Principal:** `#18181B` - Preto suave (Zinco-900) para excelente contraste em fundos brancos.
*   **Texto Secundário:** `#71717A` - Cinza neutro para rótulos, descrições e metadados de menor prioridade.

#### 4. Bordas e Divisores
*   **Borda Padrão:** `#E4E4E7` - Cinza claro para delimitação de cards, inputs e linhas de tabelas.

### Cores de Status e Contextuais

*   **Sucesso (Ativo / Saudável):** `#16A34A` (Fundo: `#DCFCE7`)
*   **Aviso / Alerta (Pausado / Atenção):** `#D97706` (Fundo: `#FEF3C7`)
*   **Erro / Falha (Crítico / Desativado):** `#DC2626` (Fundo: `#FEE2E2`)

### Cores Específicas
*   **Facebook / Meta:** `#1877F2` - Usada para integrações e status de conexão com redes de anúncios.
*   **Admin Sidebar:** `#1E1B4B` - Azul escuro profundo usado no painel ADIM para distinguir visualmente o ambiente administrativo da plataforma cliente.

### Regras de Uso
*   **Contraste:** Nunca use textos claros diretamente em superfícies brancas. Sempre utilize a combinação correta de texto primário/secundário.
*   **Sem Tons Puros:** Evite usar pretos puros (`#000000`) ou vermelhos/verdes sem saturação controlada. Use apenas os tokens especificados.
